kill在线的linux用户技巧

1.kill login user

//非法用户闯入系统
最简单的办法就是用w命令来检查,如果确认有非法用户出现在系统内，可以立即 kill 用户相关进程

kill -9  `lsof -u  xxx` ###杀掉非法用户所启动的进程
userdel -r xxx          ###删除非法用户xxx及其主目录

//正常途径kill在线用户
1.先用w命令查看该用户tty号
w
skill -kill -t pts/1  #踢出用户(-kill 可以用信号 -9代替)

2.或者先用 w 命令查看在线用户
pkill -kill -t tty  #如pkill -kill -t pts/1    //kill session
pkill -9 -t pts/1  // -9 = -kill
