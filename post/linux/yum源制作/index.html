概述

在于局域网，离线安装软件包很需要制作yum源，一般都是iso挂在制作。 如何根据rpm制作yum源?

create repo

#x01、requirement

1

yum install -y createrepo

如果不能,则下载离线的rpm在安装 rpm -ivh xxx.rpm

#x02、createrep

1
2
3
4

mkdir -p /var/repo/Packages  //创建存放rpm包的目录
cp xxx.rpm /var/repo/Packages

createrepo -v /var/repo  //生成仓库数据
