//编辑器列选择
EditPlus 
1）菜单：编辑 -> 选择 -> 列选择(Edit->Select->Column Select)
2）先按下 Alt + C ，释放，然后移动鼠标或键盘上下左右键进行列模块的移动
注意：在自动换行的模式下是不行的，改为不自动换行就行了。

Notepad++
1）按住 Alt 键，然后用鼠标进行选择 
2）按住 Alt + Shift 键，然后用箭头（键盘的上下左右箭头）进行选择
