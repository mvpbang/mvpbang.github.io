mobaxterm使用技巧

2018年8月30日
 | ssh
 | 阅读
文章目录

MobaXterm 是一款功能强大的终端仿真软件，主要用于 Windows 系统。它提供了一个多标签界面，用户可以在其中打开多个终端窗口，同时执行不同的命令。MobaXterm 支持 SSH、Telnet、Rlogin、RSFTP 等多种协议，使得远程访问服务器变得非常方便

plugin

  • https://mobaxterm.mobatek.net/plugins.html

保持ssh连接状态

mobaxterm在默认情况下，ssh长时间未使用会断开连接，使其一直连接的方法：
Settings=> SSH =>Sessions settings => 选中SSH keepalive

ModaXTerm设置右键功能

在控制台右击-> Change terminal settings -> Window -> Selection -> 右侧“Action of mouse buttons”

mobaxterm使用串口（zmodem协议）传文件

1.下载CygUtils.plugin和lrzsz插件，放到mobaxterm.exe目录
    网址：https://mobaxterm.mobatek.net/plugins.html
    插件：CygUtils.plugin，Lrzsz

2.在Linux下输入rz/sz命令

3.选择文件
    Receive file using Z-modem （接受文件，对应sz命令）  
    Send file using Z-modem     （发送文件，对应rz命令）
注意：
    此法传输比较慢，例如：波特率：115200；文件大小：1.5M（1533KB）；传输时间是：2分22秒。
    即：每秒传输大小为：10KB/S左右。

将日志实时写到指定文件

在打开的终端上右键=>"Change terminal settings" => "Terminal" => "Logging" ==>"Session logging" => "Printable output"  //将可打印的信息打印出来
"Log file name" //指定文件的位置

解决ssh窗口切换时卡顿问题

去掉"X11-Forwarding"和"Compression"选项
