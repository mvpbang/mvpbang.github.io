nginx1.9+新增tcp-udp代理模块stream

2018年12月5日
 | nginx
 | 阅读
文章目录

Nginx的stream模块用于TCP/UDP流量的代理和负载均衡，它从版本 1.9.0 开始提供，但默认情况下不是构建的，需要在编译时通过 –with-stream 参数启用 。这个模块可以处理四层协议的转发、代理或负载均衡等任务

实现TCP/UDP代理转发,目的实现，局域网对另一个网段的oracle数据库tcp代理转发

官方对stream解释

  • http://nginx.org/en/docs/stream/ngx_stream_core_module.html

编译

./configure --prefix=/servyouapp/nginx \
--with-stream \
--with-http_stub_status_module

nginx.conf部分配置

stream {
    upstream backend {
        server 1.1.1.1:1521;
    }
    server {
        listen 8686;
        proxy_connect_timeout 8s;
        proxy_timeout 24h;   #代理超时
        proxy_pass backend;
    }
}

完整配置

user  nobody;
worker_processes  1;
error_log  logs/error.log  info;

pid        logs/nginx.pid;


events {
    worker_connections  2048;
